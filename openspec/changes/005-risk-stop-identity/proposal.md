# Change: 强化风险停手游戏身份

## Why
比赛反馈认为 PV 乍看像小丑牌，需要主动建立差异化。本作不是牌型构筑游戏，而是"抽取—过热—停手"的风险停手游戏。

## What
增加危险停手奖励、超额通关奖励，并更新规则说明和 README。

## Requirements
1. 高热量停手获得最终分数奖励
2. 达标通关时根据超额比例获得额外金币
3. 规则说明第一屏必须强调"这不是出牌游戏，而是超频停手游戏"
4. README 和 docs/current-game-rules.md 必须同步更新

## Acceptance Criteria
- 热量接近上限时停手能看到明确奖励
- 超额通关能看到额外金币
- 玩家能从规则说明中理解核心技巧是"在爆炸前停手"
- 不修改无关系统

## Non-goals
- 不新增敌人
- 不新增复杂剧情
- 不重做 UI
- 不改技术栈
