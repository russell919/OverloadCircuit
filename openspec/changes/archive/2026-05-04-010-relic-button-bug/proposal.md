# Change: Bugfix - 失控回路导致按钮失效

## Why
严重BUG：获得失控回路遗物后，当热量 >= 7 时，抽取模块和停手结算按钮都会失效，即使点击重新开始仍然无法恢复。

## What
修复失控回路遗物导致的按钮失效问题：
1. 审计按钮禁用逻辑
2. 审计失控回路代码
3. 确保状态正确重置
4. 确保热量检查没有边界问题

## Requirements

### 问题分析
- 失控回路在热量 >= 7 时触发
- 可能是 processing 状态没有正确重置
- 可能是热量检查有边界条件问题
- 可能是按钮状态管理错误

### 修复目标
- 确保按钮在热量 >= 7 时正常工作
- 确保重新开始后状态正确重置
- 确保没有死循环或状态错误

## Acceptance Criteria
- 失控回路遗物存在时，热量 >= 7 的回合按钮正常工作
- 重新开始后按钮正常工作
- 没有死循环或状态错误
