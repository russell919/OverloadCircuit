📖 规则说明

游戏目标

每一关需要达到目标分数才能进入下一关。关卡会无限进行下去，目标分数会越来越高。

开局流程

主菜单 → 选择核心芯片 → 开始游戏

核心芯片将影响整局游戏的策略方向。

回合机制

  • 每关有 3 次回合机会
  • 每回合开始时，分数相关数值（筹码、倍率、X倍率）会重置为初始值
  • 热量上限为 10（缓冲核心可提升至12）
  • 你可以随时选择"停手结算"，将本回合分数加入累计
  • 如果热量达到上限（10），本回合过载，分数清零（部分遗物可保留部分分数）
  • 达标即刻通关：当本关累计分数达到目标分数时，立即通关

分数计算

本回合得分 = 筹码 × 倍率 × X倍率

金币系统

通关后获得金币奖励，公式：

金币 = 3 + 剩余回合数 × 2 + 进度奖励

  • 3：基础通关奖励
  • 剩余回合数：未使用的回合数，每回合返还 2 金币
  • 进度奖励：每 3 关额外 +1 金币

商店系统

通关后进入商店，可以用金币购买遗物：

  • 每5关出现一次商店
  • 每次商店展示 3 个可购买遗物
  • 遗物稀有度：普通（4金币）、稀有（6金币）、史诗（8金币）
  • 购买的遗物从下一关开始生效
  • 已拥有的遗物不会再次出现在商店

通关流程

达标 → 金币结算 → 选择遗物 → 商店购买 → 进入下一关

核心芯片系统

开局选择核心芯片，该选择将影响整局游戏。核心芯片不计入遗物数量。

核心芯片列表

🔋 缓冲核心
每回合热量上限 +2（原10 → 12）
作用：提高容错率和可操作空间
⚡ 增幅核心
每回合开始时，初始倍率 = 2（原1）
作用：提高前期基础输出
💎 储能核心
每回合开始时，初始筹码 = 40（原0）
作用：提高前期保底得分
☢️ 裂变核心
每回合开始时，初始X倍率 = 2（原1）
作用：提供高风险高收益开局
