# 当前游戏规则：《过载回路》

## 游戏定位
单屏、无限关卡、抽取停手、爆分构筑类游戏。

## 核心身份宣言
这不是出牌游戏。你操作的是一台不断升温的分数机器。每次抽取都会让它更强，也更接近爆炸。在爆炸前停手，就是你的核心技巧。

## 核心循环
主菜单 -> 选择开局核心芯片 -> 进入关卡 -> 抽取模块 -> 停手或过载 -> 达标通关 -> 获得金币 -> 商店购买遗物 -> 下一关 -> 失败结算。

## PVP 联机模式
主菜单点击联机模式后进入局域网搜索大厅。两名玩家通过同一台主机运行的开发服务配对，开局跳过核心芯片选择，改为 96 初始金币的全遗物自选商店。

PVP 为 1v1 对战，每关固定 3 回合且没有目标分。双方独立游玩并提交三回合成绩，服务器在双方都提交后同时公开双方三回合分数、本关总分和当前胜点。总分高者获得 1 胜点，平局双方各获得 1 胜点；当一方胜点领先 3 点时，对局结束。

PVP 每关结束后都有随机遗物奖励和随机商店。金币奖励为 16 基础金币，加上败方 8 金币补偿，以及当前胜点落后方 `8 × 落后胜点数` 金币补偿。

## 关卡规则
每关有目标分；最多 3 回合；回合结束后检查累计分是否达标；达标即刻通关；剩余回合转为金币；3 回合后未达标则失败。

## 回合规则
回合开始初始化筹码、倍率、X倍率、热量；玩家可抽取模块或停手；热量达到上限则过载。

## 分数公式
本回合得分 = 筹码 × 倍率 × X倍率。如代码实际不同，以代码为准并标注待确认。

## 结算预览系统

游戏提供精确的结算预览，帮助玩家做出明智决策：

- **当前构筑基础分**：显示当前筹码 × 倍率 × X倍率 的基础得分
- **现在停手可得**：显示点击停手后实际会获得的最终分数，包含危险停手奖励和停手触发遗物效果
- **结算分解明细**：列出所有影响最终得分的因素，每项显示名称、触发条件、影响数值

## 危险停手奖励
玩家主动停手结算时，根据当前热量/热量上限的比例给予最终分数奖励：
- 热量比例 >= 70% 且 < 80%：最终分数 +15%
- 热量比例 >= 80% 且 < 90%：最终分数 +30%
- 热量比例 >= 90%：最终分数 +60%
- 若未达到 70%，无奖励
- 过载时不触发危险停手奖励

## 超额通关奖励
达标通关时，根据本关累计分与目标分的比例给予额外金币：
- 累计分 >= 目标分 150%：额外 +1 金币
- 累计分 >= 目标分 250%：额外 +2 金币
- 累计分 >= 目标分 400%：额外 +3 金币
- 不叠加，取最高档

## 模块说明
蓝芯：增加筹码。
红芯：增加倍率。
黄芯：增加大量筹码，但额外增加热量。
冷却：降低热量。
复制：重复触发上一个非复制模块。
放大：使下一个模块额外触发。
裂变：增加 X倍率。

## 开局核心芯片
缓冲核心：提高热量上限。
增幅核心：提高每回合初始倍率。
储能核心：提高每回合初始筹码。
裂变核心：提高每回合初始 X倍率。
说明：核心芯片不是遗物，不进入遗物池。

## 遗物系统
说明遗物来自商店购买，会改变收益结构。列出当前遗物及效果；全部使用中文术语：筹码、倍率、X倍率、热量。

## 金币与商店
金币来自通关基础奖励、剩余回合返还、关卡进度奖励、超额通关奖励。金币用于购买遗物。

## 记录系统
历史最高到达关卡、历史最高单回合分、历史最高单局总分。

## 当前待优化问题
前期死亡率、遗物池数量、世界观包装、自动数值测试 Agent。
