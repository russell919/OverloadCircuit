# 过载回路 Overload Circuit

一款单屏爆分Roguelike游戏，玩家通过抽取模块积累分数，在贪心与稳健之间做出抉择。

## 运行方式

```bash
npm install
npm run dev
```

## 游戏界面

游戏采用三栏布局：
- **左侧栏**：当前关卡、回合、目标分、累计分、金币、热量条、已拥有遗物
- **中间栏**：筹码/倍率/X倍率数值、预计结算分、模块卡片、抽取/结算按钮
- **右侧栏**：待触发效果、最近抽取、事件日志

## 核心循环

```
主菜单 → 选择核心芯片 → 开始游戏
↓
每关循环：
  每回合：抽取模块 / 停手结算
  ↓
  达标即刻通关 / 3回合结束
↓
通关奖励 → 选择遗物 → 商店(每5关) → 下一关
```

## 开局核心芯片系统

游戏开始时，玩家从4个核心芯片中选择1个，该选择将影响整局游戏策略方向。

### 核心芯片列表

| 芯片名称 | 效果 | 适用场景 |
|---------|------|---------|
| 缓冲核心 | 每回合热量上限 +2（原10 → 12） | 提高容错率，适合新手 |
| 增幅核心 | 每回合开始时，初始倍率 = 2（原1） | 提高基础输出 |
| 储能核心 | 每回合开始时，初始筹码 = 40（原0） | 保证最低收益 |
| 裂变核心 | 每回合开始时，初始X倍率 = 2（原1） | 高风险高收益 |

核心芯片不计入遗物数量。

## 关卡规则

- 每关有 **3次回合机会**
- 每回合开始时，筹码/倍率/X倍率重置为初始值
- 热量上限为 **10**（缓冲核心可提升至12）
- **达标即刻通关**：累计分数达到目标时，立即通关
- 3回合全部结束仍未达标，则游戏结束

### 分数计算

```
本回合得分 = 筹码 × 倍率 × X倍率
```

### 热量系统

- 每次抽取模块热量 +1
- 热量达到上限时 **过载爆表**，本回合分数清零
- 部分遗物可在过载时保留部分分数

## 模块系统

### 模块类型

| 模块名称 | 颜色 | 效果 |
|---------|------|------|
| 蓝芯 | 蓝色 | +20~40 筹码 |
| 红芯 | 红色 | +1 倍率 |
| 黄芯 | 黄色 | +1~2 倍率，热量+1 |
| 冷却 | 绿色 | 热量 -3（最低0），+30 筹码 |
| 复制 | 紫色 | 复制上一个模块效果 |
| 放大 | 粉色 | 接下来2次模块效果翻倍 |
| 裂变 | 绿色 | +2 X倍率 |

## 遗物系统

通关后从3个遗物中选择1个获得，遗物效果在后续关卡生效。

### 遗物列表

| 遗物名称 | 稀有度 | 价格 | 效果 |
|---------|-------|------|------|
| 超导线圈 | 普通 | 4 | 每抽取第3个模块时，额外获得 +1 倍率 |
| 散热鳍片 | 普通 | 4 | 每回合第一次抽到冷却时，额外获得 +25 筹码 |
| 稳压器 | 普通 | 4 | 每回合第一次抽到黄芯时，不增加额外热量 |
| 蓝色成瘾 | 普通 | 4 | 每个蓝芯额外提供 +15 筹码 |
| 保险丝 | 普通 | 4 | 过载时，保留本回合 35% 分数 |
| 失控回路 | 稀有 | 6 | 当热量 >= 7 时，本回合获得的筹码翻倍 |
| 红色成瘾 | 稀有 | 6 | 每个红芯额外提供 +1 倍率 |
| 极限偏执 | 稀有 | 6 | 当热量 = 9 时停手结算，本回合额外获得 +2 X倍率 |
| 空转增压 | 稀有 | 6 | 如果本回合没有抽到冷却，停手时额外获得 +1 X倍率 |
| 镜像驱动 | 史诗 | 8 | 复制模块会额外再触发一次 |

### 遗物稀有度

- **普通**：4 金币
- **稀有**：6 金币
- **史诗**：8 金币

## 金币系统

通关后获得金币奖励。

### 金币公式

```
金币 = 3 + 剩余回合数 × 2 + 进度奖励
```

- 3：基础通关奖励
- 剩余回合数：未使用的回合数，每回合返还 2 金币
- 进度奖励：每 3 关额外 +1 金币

### 金币获取示例

| 关卡 | 回合 | 基础 | 剩余回合 | 进度 | 合计 |
|-----|------|------|---------|------|------|
| 第1关 | 第1回合通关 | 3 | 4 | 0 | 7 |
| 第1关 | 第2回合通关 | 3 | 2 | 0 | 5 |
| 第1关 | 第3回合通关 | 3 | 0 | 0 | 3 |
| 第4关 | 第1回合通关 | 3 | 4 | 1 | 8 |

## 商店系统

每5关通关后出现商店，可用金币购买遗物。

### 商店规则

- 每次展示 3 个可购买遗物
- 可以购买任意数量买得起的遗物
- 每个商品只能买一次
- 购买的遗物从**下一关开始生效**
- 已拥有遗物不会再次出现在商店

### 遗物稀有度分布

| 关卡阶段 | 第1格 | 第2格 | 第3格 |
|---------|-------|-------|-------|
| 1~3关 | 普通 | 普通 | 80%普通 / 20%稀有 |
| 4~6关 | 普通 | 70%普通 / 30%稀有 | 70%稀有 / 30%史诗 |
| 7+关 | 50%普通 / 50%稀有 | 60%稀有 / 40%史诗 | 30%稀有 / 70%史诗 |

## 记录系统

- **最高到达层数**：历史最高到达关卡
- **最高单回合分**：历史单回合最高得分
- **历史最高总分**：历史单局总分

## 操作说明

- **抽取模块**：消耗1热量，随机抽取一个模块
- **停手结算**：将本回合分数加入累计，进入下一回合
- **重新开始**：结束当前游戏，返回主菜单

## 技术栈

- Phaser 3：游戏引擎
- TypeScript：类型安全
- Vite：构建工具
- DOM Overlay：UI层使用HTML/CSS实现，解决文字模糊问题
