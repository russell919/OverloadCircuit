# 过载回路 Overload Circuit

一款单屏爆分Roguelike游戏，玩家通过抽取模块积累分数，在贪心与稳健之间做出抉择。

## 运行方式

```bash
npm install
npm run dev
```

## 游戏界面

游戏采用三栏布局：
- **左侧栏**：当前关卡、回合、目标分、累计分、金币、热量条、已拥有遗物
- **中间栏**：筹码/倍率/X倍率数值、预计结算分、模块卡片、抽取/结算按钮
- **右侧栏**：待触发效果、最近抽取、事件日志

## 核心循环

```
主菜单 → 选择核心芯片 → 开始游戏
↓
每关循环：
  每回合：抽取模块 / 停手结算
  ↓
  达标即刻通关 / 3回合结束
↓
通关奖励 → 选择遗物 → 商店(每5关) → 下一关
```

## 开局核心芯片系统

游戏开始时，玩家从4个核心芯片中选择1个，该选择将影响整局游戏策略方向。

### 核心芯片列表

| 芯片名称 | 效果 | 适用场景 |
|---------|------|---------|
| 缓冲核心 | 每回合热量上限 +2（原10 → 12） | 提高容错率，适合新手 |
| 增幅核心 | 每回合开始时，初始倍率 = 2（原1） | 提高基础输出 |
| 储能核心 | 每回合开始时，初始筹码 = 40（原0） | 保证最低收益 |
| 裂变核心 | 每回合开始时，初始X倍率 = 2（原1） | 高风险高收益 |

核心芯片不计入遗物数量。

## 关卡规则

- 每关有 **3次回合机会**
- 每回合开始时，筹码/倍率/X倍率重置为初始值
- 基础热量上限为 **10**，核心芯片和遗物可以继续提高上限
- **达标即刻通关**：累计分数达到目标时，立即通关
- 3回合全部结束仍未达标，则游戏结束

### 分数计算

```
本回合得分 = 筹码 × 倍率 × X倍率
```

### 热量系统

- 每次抽取模块热量 +1
- 热量达到上限时 **过载爆表**，本回合分数清零
- 部分遗物可在过载时保留部分分数

## 模块系统

### 模块类型

| 模块名称 | 颜色 | 效果 |
|---------|------|------|
| 蓝芯 | 蓝色 | +30 筹码 |
| 红芯 | 红色 | +1 倍率 |
| 黄芯 | 黄色 | +60 筹码，热量 +2 |
| 冷却 | 绿色 | 热量 -3（最低0） |
| 复制 | 紫色 | 复制上一个模块效果 |
| 放大 | 粉色 | 放大接下来1次模块抽取；若放大复制/放大，则放大待命次数 +1 |
| 裂变 | 绿色 | +1 X倍率 |

## 遗物系统

- 普通：42%

## 金币系统

通关后获得金币奖励。

### 金币公式

```
金币 = 3 + 剩余回合数 × 2 + 进度奖励
```

- 3：基础通关奖励
- 剩余回合数：未使用的回合数，每回合返还 2 金币
- 进度奖励：每 3 关额外 +1 金币

### 金币获取示例

| 关卡 | 回合 | 基础 | 剩余回合 | 进度 | 合计 |
|-----|------|------|---------|------|------|
| 第1关 | 第1回合通关 | 3 | 4 | 0 | 7 |
| 第1关 | 第2回合通关 | 3 | 2 | 0 | 5 |
| 第1关 | 第3回合通关 | 3 | 0 | 0 | 3 |
| 第4关 | 第1回合通关 | 3 | 4 | 1 | 8 |

## 商店系统

每5关通关后出现商店，可用金币购买遗物。

### 商店规则

- 每次展示 3 个可购买遗物
- 可以购买任意数量买得起的遗物
- 每个商品只能买一次
- 购买的遗物立即加入遗物列表，在后续回合/关卡中按具体效果生效
- 不可重复获得的遗物不会再次出现在商店
- 拥有保险丝后，遗物池中的保险丝会替换为强化保险丝
- 熔断协议、保险丝、量子旁路属于“至多获取一个”的遗物，规则说明界面会显示该特性

### 遗物稀有度分布

商店每个格子独立使用统一稀有度概率：

- 普通：42%
- 稀有：31%
- 史诗：20%
- 传说：7%

## 记录系统

- **最高到达层数**：历史最高到达关卡
- **最高单回合分**：历史单回合最高得分
- **历史最高总分**：历史单局总分

## 操作说明

- **抽取模块**：消耗1热量，随机抽取一个模块
- **停手结算**：将本回合分数加入累计，进入下一回合
- **重新开始**：结束当前游戏，返回主菜单

## 与小丑牌的差异

《过载回路》与《小丑牌》都强调构筑、倍率、大数字，但核心玩法有本质区别：

| 维度 | 《小丑牌》 | 《过载回路》 |
|------|-----------|-------------|
| 核心机制 | 出牌组合 | 抽取模块 |
| 风险来源 | 手牌管理 | 热量累积 |
| 策略重心 | 牌型搭配 | 停手时机 |
| 失败条件 | 手牌耗尽 | 过载爆表 |

**本作不是牌型构筑游戏，而是一款"抽取—过热—停手"的超频停手游戏。**

每次抽取模块都会增加热量，让分数变得更强，也更接近爆炸。在爆炸前停手结算，就是你的核心技巧。

### 结算预览系统

游戏提供精确的结算预览，帮助玩家做出明智决策：

- **当前构筑基础分**：显示当前筹码 × 倍率 × X倍率 的基础得分
- **现在停手可得**：显示点击停手后实际会获得的最终分数
- **结算分解明细**：列出所有影响最终得分的因素（危险停手奖励、遗物效果等）

### 危险停手奖励
在高温时停手（热量 >= 70%），可获得额外分数加成：
- 70-80%：+15%
- 80-90%：+30%
- 90%+：+60%

### 超额通关奖励
高分通关可获得额外金币：
- 累计分 >= 目标分 150%：+1 金币
- 累计分 >= 目标分 250%：+2 金币
- 累计分 >= 目标分 400%：+3 金币
